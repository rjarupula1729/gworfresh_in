# GrowFresh Geolocation-Based Products Guide

## Overview
The `products.json` file now contains **region-specific and climate-specific product availability, pricing, and recommendations** for minerals/compost products. This ensures users only see products suitable for their geographic location and soil conditions.

---

## 🌍 Three Main Regions Configured

### 1. 🏙️ HYDERABAD (Deccan Plateau, Telangana)

**Climate Profile:**
- **Season**: Semi-arid to Semi-humid
- **Soil Type**: Black soil (Vertisol) - **High pH 7.5-8.5** (Alkaline)
- **Temperature**: 32-40°C (Summer), 18-25°C (Winter)
- **Rainfall**: 558mm annually
- **Challenge**: High pH alkaline, clay-heavy, waterlogging in monsoon

**Key Soil Characteristic:**
- High clay content, high cation exchange capacity
- Tends to waterlog in monsoon
- Cracks severely in summer

**Must-Have Products:**
| Product ID | Name | Priority | Price | Why? |
|-----------|------|----------|-------|------|
| 13 | Coconut Coir Dust 3kg | **CRITICAL** | ₹129 | Water retention for summer heat |
| 14 | Wood Charcoal Powder 2kg | **HIGH** | ₹179 | pH management + drainage |
| 16 | NPK Mix 1kg | **MEDIUM** | ₹249 | Balanced nutrients |

**Products NOT Recommended:**
- None - all compost products work in Hyderabad

**Seasonal Strategy:**
- **Summer (Apr-Jun)**: Focus on Coconut Coir + Charcoal for water retention
- **Monsoon (Jul-Sep)**: Add Neem for pest control, reduce watering
- **Winter (Oct-Mar)**: Plant most crops. Use Vermicompost + NPK
- **Spring (Jan-Mar)**: Prepare for summer heat, build soil reserves

**Best Planting Season**: October-March (Winter)

**Farmer's Tip**: 
> "Black soil is your enemy in summer but friend in winter. Coconut coir is non-negotiable. Add charcoal to maintain pH balance. Your garden will thrive in winter!"

### 2. 🏜️ RAJASTHAN (Thar Desert & Semi-arid)

**Climate Profile:**
- **Season**: Arid
- **Soil Type**: Red soil (Latosol) & Sandy loam - **pH 6.5-7.5** (Slightly acidic)
- **Temperature**: 35-45°C (Summer - EXTREME), 5-25°C (Winter)
- **Rainfall**: 250-500mm annually (Scarce!)
- **Challenge**: Severe water scarcity, sandy loose soil, extreme heat, poor water retention

**Key Soil Characteristic:**
- Low organic matter, sandy texture
- High drainage BUT loses nutrients quickly
- **Critical issue**: Almost no water retention

|-----------|------|--------|

**Seasonal Strategy:**

- **Monsoon (Jul-Sep)**: Limited rainfall. Use this window to apply NPK + Vermicompost. Prepare for next winter

**Best Planting Season**: October-February (6-month winter only)

## 💡 Implementation Guide for Frontend

### 1. Filter by Region
```javascript
function getAvailableProducts(region) {
  return products.filter(p => {
    if (!p.regions || !p.regions[region]) return false;
    return p.regions[region].available === true;
  });
}
```

### 2. Show Price Adjustments
```javascript
function getRegionalPrice(product, region) {
  const basePrice = product.priceNow;
  const adjustment = product.regions[region]?.priceAdjustment || 0;
  return basePrice + adjustment;
}
```

### 3. Display Recommendations
```javascript
function getRegionalRecommendations(region) {
  return regionSpecific[region]?.recommendedMinerals || [];
}
```
