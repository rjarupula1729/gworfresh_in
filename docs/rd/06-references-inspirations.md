# 06 · References & Inspirations

Real-world ecological models that informed GrowFresh's R&D direction.
Use these when citing prior art, building plant lists for new zones, or
writing investor / patent copy.

| Reference | Origin | What we borrow |
|-----------|--------|----------------|
| **Miyawaki Dense Forests** | Akira Miyawaki (Japan) | Multi-layer native planting · rapid greening · urban heat reduction |
| **Auroville Food Forest** | Auroville, Tamil Nadu | Regenerative layered tropical food forest at human scale |
| **Solitude Farm** | Auroville | No-till natural farming · "eat what grows" philosophy |
| **Pitchandikulam Forest** | Auroville | Native ecology + medicinal plant restoration |
| **Deccan Development Society (DDS)** | Zaheerabad, Telangana | Millet-based dryland sovereignty · women-led seed banks |
| **Syntropic Agroforestry** | Ernst Götsch (Brazil) | Succession-based planting · accelerated forest building |
| **Permaculture (Mollison/Holmgren)** | Australia | Zoning concept · 7-layer food forest taxonomy |
| **FAO Home Food Security guidelines** | United Nations | Self-sufficiency-days metric framing |

## Indian region-specific plant intelligence sources

- ICAR (Indian Council of Agricultural Research) — regional crop calendars
- KVK (Krishi Vigyan Kendras) — district-level extension data
- Telangana State Horticulture Mission — terrace garden subsidy plant lists
- Pebble Garden (Auroville) — dryland regenerative tree lists
