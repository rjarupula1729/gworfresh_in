## 🎯 RECOMMENDED: Free Deployment Stack (15 minutes)

### Database: MongoDB Atlas (3 minutes)
```
1. Go to: https://mongodb.com/cloud/atlas
2. Create free account
3. Create M0 (free) cluster
4. Save connection string:
   mongodb+srv://username:password@cluster.mongodb.net/growfresh?...
5. ✅ DONE
```

### Backend: Railway.app (5 minutes)
```
1. Go to: https://railway.app
2. Click "New Project"
3. Select "Deploy from GitHub"
4. Choose: rjarupula1729/gworfresh_in
5. Add MongoDB addon (built-in)
6. Deploy
7. Get Backend URL: https://your-project.up.railway.app
8. ✅ DONE
```

### Frontend: Vercel (5 minutes)
```
1. Go to: https://vercel.com
2. Click "Import Project"
3. Select: rjarupula1729/gworfresh_in
4. Click "Import"
5. Deploy
6. Get Frontend URL: https://gworfresh-in.vercel.app
7. ✅ DONE
```

### Connect (2 minutes)
```
1. Update frontend .env:
   REACT_APP_API_URL=https://your-project.up.railway.app
2. Redeploy frontend (automatic on Vercel)
3. ✅ DONE - YOUR APP IS LIVE!
```

## 🆘 TROUBLESHOOTING

### If Frontend won't deploy
```
Check:
1. Build logs in Vercel dashboard
2. Dependencies in package.json
3. Environment variables set
4. No syntax errors
```

### If Backend won't deploy
```
Check:
1. Deployment logs in Railway
2. MongoDB connection string
3. Environment variables
4. Port configuration
```

### If Images not showing
```
Check:
1. Image URLs are valid
2. Seeding script ran
3. Products have 'image' field in DB
4. No CORS issues
```

### If APIs not responding
```
Check:
1. Backend is running
2. Database is connected
3. Environment variables correct
4. Check backend logs
```
